# Trabalho Prático - Java POO

## Descrição

Este repositório contém os projetos desenvolvidos na disciplina de Programação Orientada a Objetos utilizando a linguagem Java.

## Projetos

### CadastroPOO

Sistema para cadastro de Pessoas Físicas e Pessoas Jurídicas utilizando:

* Programação Orientada a Objetos (POO)
* Herança
* Polimorfismo
* Serialização
* Persistência de dados em arquivos binários

### CadastroPOO_2

Evolução do projeto anterior com cadastro em modo texto, permitindo:

* Inclusão de registros
* Alteração de registros
* Exclusão de registros
* Consulta por ID
* Listagem de registros
* Salvamento em arquivos binários
* Recuperação de dados persistidos

## Tecnologias Utilizadas

* Java
* Apache NetBeans
* Git
* GitHub

CadastroPOO_2/
